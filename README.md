# MLOPS-Jenkins-Shared-Library-CI-CD-Project
This project showcases a professional MLOps workflow by centralizing automation in a Jenkins Shared Library, ensuring standardized, reusable builds and deployments. The pipeline automates Docker image creation and Kubernetes orchestration on a GCP VM, transforming complex manual tasks into a streamlined, scalable process.
